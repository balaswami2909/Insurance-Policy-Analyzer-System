# Insurance Policy Analyzer System

### Tech Stack
- Java
- JDBC
- MySQL
- Spring Boot (Basic)
- Maven
- Console UI

### Instructions
1. Import into Eclipse as Maven Project
2. Run `schema.sql` to set up DB
3. Set DB user/password in `application.properties`
4. Run `InsurancePolicyAnalyzerApplication.java`
